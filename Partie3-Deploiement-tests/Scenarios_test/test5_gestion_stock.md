# Test 5 — Consulter le stock

## Informations générales

| Champ | Détail |
|-------|--------|
| Module | Inventaire |
| Fonctionnalité testée | Consultation des niveaux de stock en temps réel |
| Chemin | Inventaire → Produits |
| Date d'exécution | 23/03/2026 |
| Statut | Réussi ✅ |

## Objectif

Vérifier que le module Inventaire affiche les niveaux de stock en temps réel pour chaque produit du catalogue.

## Résultats observés

| Produit | Quantité en stock |
|---------|------------------|
| Ordinateur portable HP | 15 unités |
| Souris sans fil Logitech | 50 unités |
| Clavier USB Dell | 40 unités |
| Écran 24 pouces Samsung | 10 unités |
| Disque dur externe 1To | 25 unités |

## Étapes réalisées

1. Aller dans le module **Inventaire**
2. Cliquer sur **Produits** dans le menu du haut
3. Observer les niveaux de stock affichés sous chaque produit

## Résultat obtenu

Les niveaux de stock de chaque produit sont visibles directement dans la liste du catalogue. L'information est disponible en temps réel sans nécessiter de rapport manuel, contrairement à l'ancienne gestion par fichiers Excel de TechDistrib SARL.
